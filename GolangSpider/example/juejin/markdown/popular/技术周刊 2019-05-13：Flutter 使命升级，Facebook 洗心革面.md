### 前端快爆 ###

* Flutter 近期动作频频， 发布了 1.5 版本，提供了 Android 和 iOS 的内购插件，升级到了 iOS 12.1 SDK 以响应苹果的政策……但最激动人心的莫过于 [Flutter for web]( https://link.juejin.im?target=https%3A%2F%2Fmedium.com%2Fflutter%2Fbringing-flutter-to-the-web-904de05f0df0 ) 的发布，这使得 Flutter 程序可以直接运行在任何浏览器上( [Demo]( https://link.juejin.im?target=https%3A%2F%2Fwww.nytimes.com%2Fgames%2Fprototype%2Fkenken%23%2F ) )，同时 Flutter 团队把自己的使命扩展成“ [打造在任何屏幕上创造美好体验的最佳框架]( https://link.juejin.im?target=https%3A%2F%2Fdevelopers.googleblog.com%2F2019%2F05%2FFlutter-io19.html%3Fm%3D1 ) ”。 [🔗]( https://link.juejin.im?target=https%3A%2F%2Fmedium.com%2Fflutter%2Fbringing-flutter-to-the-web-904de05f0df0 )

> 
> 
> 
> 点评：此时的 Flutter 亮出了自己的全部野心。
> 
> 

* 

[Google I/O 2019]( https://link.juejin.im?target=https%3A%2F%2Fevents.google.com%2Fio%2F ) 召开，Chrome 开发者团队在 YouTube 上新建了合集 [Web at Google I/O 2019]( https://link.juejin.im?target=https%3A%2F%2Fwww.youtube.com%2Fplaylist%3Flist%3DPLNYkxOF6rcIATmAmz7HcCzongGvQEtx8i ) ，囊括了此次大会有关 Web 技术的所有讲座视频，值得一看。 [🔗]( https://link.juejin.im?target=https%3A%2F%2Fwww.youtube.com%2Fplaylist%3Flist%3DPLNYkxOF6rcIATmAmz7HcCzongGvQEtx8i )

* 

Chrome 74 发布，其中包括：类私有字段的使用、用 ` prefers-reduced-motion` 媒体查询减少动效的使用、transition events 的支持。 [🔗]( https://link.juejin.im?target=https%3A%2F%2Fdevelopers.google.com%2Fweb%2Fupdates%2F2019%2F04%2Fnic74%3Futm_source%3Dfeed%26amp%3Butm_medium%3Dfeed%26amp%3Butm_campaign%3Dupdates_feed )

* 

Chrome 计划于 Picture-in-Picture V2 中支持弹出任意元素，这样网站将能把网页做得更像桌面程序，当前的 Picture-in-Picture V1 仅支持视频元素的弹出。 [🔗]( https://link.juejin.im?target=https%3A%2F%2Fgithub.com%2FWICG%2Fpicture-in-picture%2Fblob%2Fv2%2Fv2_explainer.md )

* 

在 [Microsoft Build 2019]( https://link.juejin.im?target=https%3A%2F%2Fwww.microsoft.com%2Fen-us%2Fbuild ) 上 Edge 浏览器宣布了一些新功能，其中包括能收藏网页信息的 Collections、保护用户隐私的 Privacy tools、与 Chrome 同源的开发者工具以及 IE 模式。 [🔗]( https://link.juejin.im?target=https%3A%2F%2Fblogs.windows.com%2Fmsedgedev%2F2019%2F05%2F06%2Fedge-chromium-build-2019-pwa-ie-mode-devtools%2F%23FDcQqxrlJbu0DvDO.97 )

> 
> 
> 
> 点评： 最终，iPhone 添加了双卡双待，Edge 活成了国产双核浏览器的样子。
> 
> 

* VS Code 新添加 Remote Development 特性，使得开发者可以在远程环境下开发、调试。 [🔗]( https://link.juejin.im?target=https%3A%2F%2Fcode.visualstudio.com%2Fblogs%2F2019%2F05%2F02%2Fremote-development )

> 
> 
> 
> 点评：又少了一个用 Vim 的理由。
>

### 全民目击 ###

去年深陷各种官司的 Facebook 在 F8 大会上宣布要“洗心革面”，进行网站和 App 的大改版。全新的设计语言被称为 FB5，在 FB5 中，FB 的经典蓝更多地被白色取代，主要信息更加突出。从使用率最高的导航 Tab 可以看出群组和市场是 Facebook 接下来关注的重心。 [🔗]( https://link.juejin.im?target=https%3A%2F%2Ftechcrunch.com%2F2019%2F05%2F01%2Faspirationbook%2F )
